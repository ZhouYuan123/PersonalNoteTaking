# C 语言

## 1. 简介

助记符，汇编语言。

计算机语言：人和计算机交流的语言。

1972年上线，面向过程，由B语言发展而来，贝尔实验室。

国际标准：C89/C90

广泛应用于底层软件开发(OS, 驱动)。

```c
#include <stdio.h>
int main()
{
	printf("Hello World\n");
	return 0;
}
```

## 4. 字符串

双引号。隐式最后一个字符是`\0`, 这是字符串的结束标志。

char arr[] = “hello”; or char arr[] = {'a', 'b' ,'c', '\0'};

```c
#include <string.h>
int len = strlen("abc"); // 3
```

`??)` : 三字母词，等于 ]

`\a` : 响铃

`\123` : 跟一至三个八进制数字

`\x12`: 跟两个十六进制数字
